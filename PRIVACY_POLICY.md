# Privacy Policy for Calculator Game

**Last Updated:** [DATE]

## Overview

Calculator Game ("we", "our", or "the app") is a multiplayer estimation game. This privacy policy explains how we handle information when you use our mobile application.

## Information We Collect

### Information You Provide
- **Nickname:** A display name you choose when joining a game. This is not linked to your real identity.
- **Game Questions:** Questions you create during gameplay.

### Information Collected Automatically
- **Anonymous Identifier:** We use Firebase Anonymous Authentication to create a temporary, anonymous user ID for game synchronization. This ID is not linked to your personal identity.
- **Game Data:** Your guesses, scores, and game participation data stored temporarily during active games.
- **Device Information:** Basic device information for app functionality and crash reporting (via Sentry).

## How We Use Information

We use the collected information solely to:
- Enable real-time multiplayer gameplay
- Synchronize game state between players
- Validate questions using AI (Google Gemini)
- Improve app stability through crash reporting
- Maintain game sessions

## Data Storage & Retention

- **Game Data:** Stored in Firebase Realtime Database only during active game sessions. Game data may be automatically deleted after games end or become inactive.
- **Anonymous IDs:** Temporary and not linked to personal information.
- **No Account Creation:** We do not require or store email addresses, passwords, or personal accounts.

## Third-Party Services

Our app uses the following third-party services:

### Firebase (Google)
- Purpose: Anonymous authentication and real-time game data synchronization
- Privacy Policy: https://firebase.google.com/support/privacy

### Google Gemini AI
- Purpose: Question validation and answer generation
- Privacy Policy: https://policies.google.com/privacy
- Note: Questions submitted for validation are processed by Google's AI. No personal information is included.

### Sentry
- Purpose: Error tracking and crash reporting
- Privacy Policy: https://sentry.io/privacy/

## Data Sharing

We do not:
- Sell your information to third parties
- Share personal information with advertisers
- Use your data for marketing purposes
- Create user profiles for tracking

Game data is only shared with other players in your game session for gameplay purposes.

## Children's Privacy

Calculator Game does not knowingly collect personal information from children under 13. The app uses anonymous authentication and does not require any personal information to play.

## Your Rights

Since we collect minimal, anonymous data:
- No personal account to delete
- No personal information stored
- Game data is temporary and session-based

If you have concerns about any data, contact us at the email below.

## Security

We implement appropriate security measures including:
- Encrypted data transmission (HTTPS/TLS)
- Firebase security rules to protect game data
- No storage of sensitive personal information

## Changes to This Policy

We may update this privacy policy from time to time. We will notify users of significant changes through the app or app store listing.
